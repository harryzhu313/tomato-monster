<h1 align="center">Monster pomodoro · 按时停下来</h1>

<p align="center">解决番茄钟最大的问题：时间到了，却停不下来。</p>

---

## Why — 它解决什么问题

市面上的番茄钟一直用不起来。分析下来，是我的两个问题没被好好解决：

### 1. 25 分钟结束后，根本停不下来

传统番茄钟时间到了只会弹一个小通知，一忙起来就顺手忽略。然后「再写 5 分钟就停」变成半小时、一小时，节奏越拖越乱，专注力反而越来越差。

**Tomato Monster 的做法是强制干预**：25 分钟一到，整个屏幕被全屏锁屏页覆盖，你没办法继续打字、切窗口。唯一的出路是消耗当天有限的「加时额度」（默认每天 3 次）—— 用物理层的阻断代替薄弱的意志力，强制你起身、看远、喝水。

> ⚠️ 目前只能在浏览器层面实现锁屏，系统层面的强制锁屏还没做到，后续还需要继续开发。

### 2. 每日任务记录无处沉淀

今天完成了几个番茄、做了什么任务？事后想回顾，只能靠记忆。手动把这些数据写进 Notion，又麻烦到谁都坚持不了。

**Tomato Monster 会把每日任务记录自动导出到你自己的 Notion 数据库**：任务名、日期、计划/实际番茄数、分类等字段一键同步。数据完全在你自己的空间里，积累越久越有价值。

---

## What — 功能特点

- **核心计时**：经典 25 / 5 分钟节奏，休息时自动全屏锁屏（可选背景：晨曦 / 海洋 / 黄昏）
- **加时额度**：每天 3 次加时机会，用完即止，戒掉「再来一会儿」的惯性
- **任务管理**：添加任务、选择分类（工作 / 学习 / 生活 / 兴趣爱好）、设定计划番茄数、标记当前专注任务
- **每日收获**：🍅 成熟番茄（已完成）与 🥫 烂番茄（中途放弃）可视化展示
- **连击与热力图**：连续天数徽章、7 天成就动画、仿 GitHub 52 周年度热力图
- **怪兽表情主题**：Happy / Calm / Angry / Afraid / Love 五种状态，随专注、休息、威胁场景自动切换
- **桌面悬浮窗**：支持 Document Picture-in-Picture（Chrome 116+），计时器永远置顶不被遮挡
- **日末提醒**：到点如果今天还没把任务导入 Notion，会发一条系统通知提醒你（待开发）
- **Notion 导出**：每日任务一键同步到你自建的数据库
- **本地优先**：所有数据保存在 `chrome.storage.local`，Token 不上传任何服务器

## How — 安装与配置

### 一、安装到 Google Chrome

目前还没上架 Chrome Web Store，使用开发者模式加载即可：

1. 下载本仓库：`git clone` 本项目，或点 GitHub 右上角「Code → Download ZIP」后解压
2. 打开 Chrome，地址栏输入 `chrome://extensions/` 回车
3. 打开右上角的「**开发者模式**」开关
4. 点左上角「**加载已解压的扩展程序**」，选择本项目根目录（包含 `manifest.json` 的那一层）
5. 看到工具栏出现番茄图标，就装好了。建议点一下图钉把它固定在工具栏

### 二、配置 Notion 导出（零基础详细版）

> 先理解一下原理：Notion 的外部程序要写入数据，必须借助一个叫「Integration（集成）」的东西。流程是：
> **创建 Integration → 拿到 Token → 创建目标数据库 → 把数据库「分享」给 Integration → 把 Token 和数据库 ID 填进扩展**。
>
> 免费账号完全可用，不需要付费。

#### 步骤 1：创建 Integration，拿到 Token

1. 浏览器访问 <https://www.notion.so/my-integrations>
2. 点击左上角「**+ New integration**」
3. 填写：
   - **Name**：任意，例如 `Tomato Timer`
   - **Associated workspace**：选择你要用的工作空间
   - **Type**：保持默认 `Internal`
4. 点 Save
5. 进入 Integration 详情页，找到「**Internal Integration Token**」一栏，点「Show」→ 复制
6. Token 形如 `ntn_xxxxxxxxxxxxxxxx`，先贴到记事本里备用

> 🔒 Token 只会保存在你本地，任何时候都可以回 Notion 后台 Revoke 掉。

#### 步骤 2：创建「任务数据库」

> 🚀 **推荐：直接复制现成模板**
>
> 为了让你少走弯路，我准备好了一份开箱即用的 Notion 模板，属性和选项都已按扩展的要求配好：
>
> 👉 [**点我打开模板页**](https://fortune-iron-d66.notion.site/template-40e62405089a82208b3181d8fbbc7517)
>
> 打开后点右上角「**Duplicate**」就会复制到你自己的工作空间。然后跳过下面的手动建表，直接去步骤 3。

如果你想自己手动建，也可以：在 Notion 中新建一个页面，正文输入 `/database` → 选择「**Database - Full page**」。

为这个数据库配置以下属性（**名称必须完全一致**，类型要严格匹配）：

| 属性名 | 类型 | 说明 |
| --- | --- | --- |
| 任务名 | Title | 把默认的 Name 列改成这个名字 |
| 日期 | Date | 任务所属日期 |
| 计划番茄 | Number | 当天计划几个番茄 |
| 实际番茄 | Number | 实际完成的数量 |
| 超额番茄 | Number | 超过计划完成的部分 |
| 放弃番茄 | Number | 中途放弃（烂番茄）数 |
| 分类 | Select | 选项：`工作` / `学习` / `生活` / `兴趣爱好` |

#### 步骤 3：把数据库分享给 Integration（⚠️ 最容易卡住的一步）

1. 打开刚才创建的数据库页面
2. 点右上角「**···**」菜单
3. 下拉找到「**Connections**」（老版本叫「Add connections」）
4. 搜索并选中步骤 1 创建的 Integration 名字
5. 点 Confirm 确认授权

> 如果后面测试时报 401 或 unauthorized，99% 是这一步漏了。

#### 步骤 4：复制数据库 ID

1. 在数据库页面点右上角「**Share → Copy link**」
2. 链接形如：
   ```
   https://www.notion.so/workspace/23abc456def7890...?v=...
   ```
3. `workspace/` 之后、`?` 之前的那一段就是 **Database ID**（32 位字符，含或不含连字符都可以）

#### 步骤 5：填进扩展设置

1. 在 Chrome 工具栏右键点击番茄图标 → 选「**选项**」，或从弹窗内进入设置页
2. 找到「**📤 Notion 导出**」区域，依次填入：
   - **Internal Integration Token**：粘贴 `ntn_xxx`
   - **任务 DB ID**：粘贴步骤 4 的 32 位 ID
   - **日页面 DB ID**（可选，留空即可不用）
3. 点「**测试**」按钮，看到成功提示就 OK 了
4. 回到扩展弹窗 → 打开「历史」→ 找某一天点「导入 Notion」，试试第一次同步

#### 可选：按「日」聚合

想让每天的任务在 Notion 里关联到一个「日页面」？
1. 再建一个数据库，包含一个 `日期`（Date 类型）属性
2. 把它也按步骤 3 分享给同一个 Integration
3. 在「任务数据库」里新增一个 Relation 属性，指向这个日数据库
4. 把新数据库的 ID 填到扩展设置的「日页面 DB ID」

#### 常见问题

- **测试报 401 / unauthorized**：多半是 Integration 没在数据库的 Connections 里添加，或 Token 复制漏了字符
- **报 property not found**：数据库属性名或类型对不上，按上方表格逐项检查
- **免费账号能用吗**：可以，Internal Integration 不需要付费
