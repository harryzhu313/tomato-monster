📖 历史明细

每一天的任务：计划 vs 实际。为后续导入 Notion 做准备。

📤 Notion 导出

把每天的任务作为行记录导入 Notion。导入格式：每个任务一行，映射"任务名 / 日期 / 番茄钟数 / 分类"；若填了日页面 DB ID，会自动按日期关联到对应的"日"页面。

Token 只存在本机（chrome.storage.local），不会上传到任何服务器。 前往 Notion Integrations 生成 token， 并把 DB 分享给该 integration（DB 右上 ···→Connections）。

🔔 通知

关掉提示音只影响本扩展，不影响系统其他通知音效。

🪟 桌面悬浮

popup 里点「🪟 悬浮」会单独弹出一个小窗口。关掉这个 popup，小窗还在；把它拖到屏幕任意角落即可随时看计时。

想置顶浮在所有应用之上？点悬浮窗里的 📌 按钮。 Chrome 116+ 原生支持 Document Picture-in-Picture，点一下 📌 就会弹出一个系统级置顶的小窗，切应用、切桌面它都在最上层。不需要任何第三方工具。

若 Chrome 版本较老，可用系统工具兜底：macOS 的 AfloatX / Hammerspoon，Windows 的 PowerToys Always On Top。
想让番茄在关掉所有 Chrome 窗口后还继续响？ 需要在 Chrome 设置里开「Chrome 关闭后继续运行后台应用」。

开启后，即使关掉所有 Chrome 窗口，系统通知和计时器仍会响；但悬浮小窗本身是一个窗口，手动关掉它就是关掉了。
